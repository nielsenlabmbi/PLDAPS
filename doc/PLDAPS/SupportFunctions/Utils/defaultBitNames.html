defaultBitNames

PURPOSE

dv = defaultBitNames(dv)

SYNOPSIS

function dv = defaultBitNames(dv)

DESCRIPTION 

 dv = defaultBitNames(dv)
 defaultBitNames adds .events.NAME to dv
 The MAP server can only take 7 unique bits. BIT names are:
   1. FIXATION
   2. STIMULUS
   3. TARGS
   4. REWARD
   5. BREAKFIX
   6. TRIALEND
   7. TRIALSTART
