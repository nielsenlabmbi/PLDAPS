@pldaps/beginExperiment

PURPOSE

beginExperiment initialize an Exeriment

SYNOPSIS

function p = beginExperiment(p)

DESCRIPTION 

beginExperiment    initialize an Exeriment
 p = beginExperiment(p)
 initialize the beginning of time for this experiment
 beginExperiment checks which devices are connected to PLDAPS and gets
 timestamps from each of them and stores them in the devives substructures
 of p.defeaultParameters

CROSS-REFERENCE INFORMATION

This function calls:
This function is called by:
  • @pldaps/replay replay replay a previously recorded experiment on screen
  • @pldaps/run run run a new experiment for a previuously created pldaps class

  • .datapixx.use: 24 enable control of VPixx devices
  • .eyelink.use: 28 use the eyelink moduke
  • .pldaps.version.logo: 18

This function assigns these parameters:
  • .datapixx.experimentStartDatapixx: 25

  • .pldaps.version.number: 16
