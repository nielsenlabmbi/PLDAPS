pds.plexon.spikeserver.connect

PURPOSE

pds.plexon.spikeserver.connect connect to a remote plexon spikeserver

SYNOPSIS

function p = connect(p)

DESCRIPTION 

pds.plexon.spikeserver.connect   connect to a remote plexon spikeserver

 connects to the spikeserver and requests to get the file currently
 recorded to.

 p = pds.plexon.spikeserver.connect(p)

PARAMETER USAGE INFORMATION

This function uses these parameters:
  • .plexon.spikeserver.continuous: 33 true if you want data to be sent when it arrived
  • .plexon.spikeserver.eventsonly: 28 true if you only want to receive events, not spikes
  • .plexon.spikeserver.isConnected: 40
  • .plexon.spikeserver.remoteip: 19 29 34 38 IP of the plexon machine
  • .plexon.spikeserver.remoteport: 19 29 34 38 Port set on the plexon machine's spikeserver
  • .plexon.spikeserver.selfip: 14 19 This machine's IP
  • .plexon.spikeserver.selfport: 19 The port this machine should use
  • .plexon.spikeserver.use: 10 toggle use of our plexon spikeserver
