Index for .datapixx.adc

PURPOSE

continuously collect and store adc data from Datapixx.

DESCRIPTION 

continuously collect and store adc data from Datapixx.
  Timestamps for each sample are stored in datapixx.adc.dataSampleTime. The number of collect samples is in datapixx.adc.dataSampleCount
