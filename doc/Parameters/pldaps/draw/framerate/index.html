Index for .pldaps.draw.framerate

PURPOSE

control drawing of a framerate history to see framedrops.

DESCRIPTION 

control drawing of a framerate history to see framedrops.
  If drawn, expected frame interval (ifi) is marked as a solid black line, 0 and 2*ifi as dashed lines. Actual ifi as red dots.
