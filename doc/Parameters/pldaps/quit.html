Index for .pldaps.quit

PURPOSE

control expeiment during a trial.

DESCRIPTION 

control expeiment during a trial.
  1: go into a pause mode immediately and allow paraneters to be changed for future trials. Any changes to parameters will overrule previous parameters not defined in a condition struct. 2: end experiment immediately. If you want to end the experiment to end after the regular end of the trial, set .pldaps.finish=.pldaps.iTrial instead. When using pldapsDefaultTrialFunction, pressing p sets quit to 1 (pause) and q sets quit to 2 (end experiment)
